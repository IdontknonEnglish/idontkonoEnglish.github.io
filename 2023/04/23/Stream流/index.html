Stream流

一、获取Stream流

  1. 根据类型不同获取方法有4中

    1. 单列集合如 List 和 Set 可以直接通过 .stream()获取

    2. 双列集合如 Map 需要先转化为单例再获取 如 map.keySet().stream()

    3. 数组 直接通过工具方法获取 如 Arrays.stream(数组名)

    4. 零散数据如 {1,2,3,4,5,6} , 可以通过Stream获取 例如 Stream.of({1,2,3,4,5,6})

              1. 如果是引用类型例如String 可以将其先转化为数组再进行第三步操作

三、Stream流的终结方法

  1. toArray()的参数是数组的长度
  2. toArray()的底层是遍历流里面的元素，将他们保存到数组中
  3. toArray()的返回值是一个装着流里面所有数据的数组
