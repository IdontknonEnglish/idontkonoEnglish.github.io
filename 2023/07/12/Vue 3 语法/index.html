Vue 3 语法

响应式数据

  • 在 Vue 3 中，ref 是 Composition API 中的一个函数，用于创建一个可响应的引用值，并提供了访问和修改该引用值的方式。ref 创建的引用值可以在模板或组件中使用，并且会自动追踪其依赖关系
  • reactive 函数也用于创建响应式的数据，但与 ref 不同，它可以用于将整个对象转换为响应式对象，而不仅仅是单个值。在 reactive 中，对象的所有属性都会成为响应式的，当任何属性发生变化时，相关的界面也会自动更新。
  • 总结 普通类型数据用 ref，数组和对象用 reactive
