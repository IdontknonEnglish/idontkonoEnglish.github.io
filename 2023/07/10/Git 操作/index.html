命令字典

  1. git init：在当前目录初始化一个新的 Git 仓库。
  2. git clone <url>：从远程仓库克隆一个副本到本地。
  3. git status：查看当前工作区和暂存区的状态。
  4. git add <文件>：将文件的修改或新增操作添加到暂存区。
  5. git commit -m "<提交信息>"：将暂存区的内容提交到本地仓库。
  6. git pull <远程仓库> <分支>：从远程仓库拉取最新的更改并合并到当前分支。
  7. git push <远程仓库> <分支>：将当前分支的提交推送到远程仓库。
  8. git branch：列出本地分支，显示当前所在分支。
  9. git checkout <分支>：切换到指定的分支。
  10. git merge <分支>：将指定分支的更改合并到当前分支。
  11. git stash：将当前工作区的修改暂存起来，以便切换到其他分支。
  12. git log：查看提交记录日志。
  13. git diff：查看文件或提交之间的差异。
  14. git reset：撤销提交或将文件从暂存区移出。
  15. git remote：管理远程仓库相关操作，如添加、删除、重命名等。
