HTTP 协议 客户端与服务端交互流程

流程图

Http 协议 超文本传输协议 由万维网制定（w3c）

是浏览器和服务器通讯的应用层协议，规定了浏览器与服务器之间的交互规则以及交互数据的格式

Http协议对于客户端与服务端之间的交互规则有以下定义：

​ 要求浏览器与服务端之间必须遵循一问一答的规则， 既浏览器与服务端建立TCP连接后需要要求浏览器先发送一个请求(问)

然后服务端接收到请求并予以处理后再发送响应(答).注意，服务端永远不会主动给浏览器发送消息。

要求浏览器与服务端的传输层协议必须的可靠的传输，因此是使用TCP协议作为传输层协议的

Http 协议 对于浏览器与服务端之间的交互的数据格式，内容也有一定的要求。

浏览器发送给服务端的内容称为请求Request

服务端给浏览器发送的内容称为响应Response (这个是浏览器创建的提供给客户端返回响应的空间)
