RESTful 风格

@RequestParam 注解

作用 用于从前端接收 HTTP 请求参数。参数直接是在请求的 后面 既 ? 后面的参数

如果不指定 @RequestParam 注解，则 Spring MVC 会根据参数的名称直接从请求参数中获取值，这种方式称为默认方式或者隐式注解。 （前端参数名字和后端接收名字一样会自动对应）

如果要将?后面的参数映射到一个java对象上可以简单的省略注解或者使用@ModelAttribute不推荐后面这种

显式注解

@PathVariable 注解

@RequestBody 注解

@RequestBody 注解可以用于接收请求的 HTTP 请求体，可以将请求体转换为 Java 对象（例如 JSON 或 XML）
